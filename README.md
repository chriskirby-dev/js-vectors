# js-vectors
Vanilla JS JavaScript Vector Classes with extra methods and utilities

## Usage

### Vector2D Class

```javascript
const vector = new Vector2D(x,y);
```
or 

```javascript
const vector = new Vector2D(<vector2d>);
```

or 

```javascript
const vector = new Vector2D([x,y]);
```

### Vector3D Class

```javascript
const vector = new Vector3D(x,y,z);
```
or 

```javascript
const vector = new Vector3D(<vector3d>);
```

or 

```javascript
const vector = new Vector3D([x,y,z]);
```
